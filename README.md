# Análisis PIB vs Inflación Colombia 2010-2024 (Datos DANE)

Dashboard en Excel con datos oficiales DANE (Cuentas Nacionales Anuales).

## Insights clave
- Crecimiento promedio PIB anual 2010-2024: ~3.56% (calculado de series encadenadas base 2015).
- Año dorado post-pandemia: 2021 (+11.08% rebote).
- Inflación subyacente alta en 2022-2023 (~9-10%).
- Pregunta central: ¿Creció Colombia o solo subieron los precios? → PIB real creció, pero inflación post-pandemia erosionó ganancias.
- Efecto pandemia: 2020 (-3.3%), recuperación fuerte 2021.
